If you are reading this, it means you have a very niche requirement

To use this program, please follow these simple steps:

1. Make a directory or folder somewhere to store the program
2. Save the excel spreadsheet as a csv with the following options to the same directory of the program
	field delimiter:','	string delimiter:'"'
Note that the apostrophes are NOT included, use only whats within the apostrophes
3. run the program, and when prompted type the name of the file, including the .csv
4. the program should then create a file, named results.csv, which can be opened in excel, using the same delimiter options as above
